📊 Olist E-commerce BI Analysis

Progetto di Business Intelligence basato sul dataset pubblico di Olist, relativo agli ordini (2016–2018) di seller operanti nei marketplace brasiliani.

L’obiettivo è la realizzazione di un report interattivo per analizzare:

andamento degli ordini e dei ricavi nel tempo per stato
confronto anno su anno con variazione percentuale
distribuzione dei rating clienti

Il progetto include:

modellazione dati in star schema
utilizzo di dimensione calendario
ottimizzazione del dataset
dashboard con focus su UX, filtri e drill-through

Dataset utilizzati: ordini, prodotti, clienti, recensioni e dettagli di vendita.
